## 🧠 Descripción General

Este repositorio documenta mi proceso de aprendizaje de bases de datos, enfocándose en teoría, ejercicios prácticos y proyectos guiados. El objetivo es proveer un camino de estudio claro, completo y aplicable para cualquier persona que busque dominar conceptos esenciales de bases de datos desde cero.

<br>

## 🗂️ Estructura del Repositorio

**Este repositorio está organizado para optimizar la navegación pedagógica:**

```plain text
📁 /docs/        → Material teórico y recursos del curso
📁 /notes/       → Apuntes personales y explicaciones resumidas
📁 /exercises/   → Ejercicios prácticos con soluciones
📁 /projects/    → Proyectos reales y retos aplicados
📄 LICENSE       → Licencia de uso (MIT)
📄 README.md     → Documentación principal (este archivo)
```

Cada carpeta representa un nivel de aprendizaje, desde la teoría hasta la aplicación práctica.

## 🚀 Cómo Empezar

**Clona el repositorio y comienza por la carpeta de teoría:**

```bash
git clone https://github.com/leonardo-adames/database-learning.git
cd database-learning
```

**Desde ahí, abre las carpetas:**

```plaun text
docs/ → teoría

exercises/ → ejercicios paso a paso

projects/ → proyectos reales para practicar

(Puedes usar W3resource y WeCreateProblemn para reproducir los ejercicios.)
```
